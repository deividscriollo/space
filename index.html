How it works

This skin is created as addon component of bootstrap so as to integrate easily. You just add bootstrap-md.min.css and bootstrap-md.min.js to <head> and <body> accordingly in your bootstrap html file. 


<head>
  <link rel="stylesheet" href="css/bootstrap.min.css">
  <link rel="stylesheet" href="css/bootstrap-md.min.css">
</head>
...
...
<body>
  <script src="js/bootstrap.min.js"></script>
  <script src="js/bootstrap-md.min.js"></script>
</body>

bootstrap-md.min.css contains all classes with prefix bmd-* so that it is easy to add or remove material design stuff to or from element in your markup. For example, this button will raise up on mouseover and take ripple effect on click by adding class bmd-ripple.

Customization

If you need to custom this skin, you are strongly recommended to use {LESS} because the source file contains very useful mixins to generate css style. Please follow below steps to generate CSS stylesheets.

  1. Replace two original bootstrap files with 'bootstrap.less' and 'variables.less' from your download.
  2. Copy 'bootstrap-md.less' and 'material-design-color.less' to the same folder of 'bootstrap.less'.
  3. Compile 'bootstrap.less' file to generate 'bootstrap.min.css' by using LESS compiler.
  4. Compile 'bootstrap-md.less' file to generate 'bootstrap-md.min.css' by using LESS compiler.
